Stackable Operator for Trino

This is an operator for Kubernetes that can manage Trino clusters.

This operator only works with images from the Stackable repository

Supported Versions

The Stackable Operator for Trino currently supports the following versions of Trino:

  • 377

  • 387

  • 395

  • 396

Get Docker image

docker pull docker.stackable.tech/stackable/trino:<version>
